# Sample Files

Place your sample files here to showcase the quality of generated resources:

- `sample-presentation.pptx` - Sample PowerPoint presentation
- `sample-audio.mp3` - Sample podcast-style audio
- `sample-worksheet.docx` - Sample student worksheet
- `sample-answers.pdf` - Sample answer key

These files will be downloadable from the landing page's "Samples" section.

## Creating Samples

You can generate sample files by:
1. Creating a test lesson package in the app
2. Downloading the generated files
3. Renaming them to match the sample file names above
4. Placing them in this directory

Make sure the samples showcase high-quality content that represents what users can expect.
